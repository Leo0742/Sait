# 🚀 Быстрый старт с Docker

## Что нужно сделать

### 1️⃣ Установите Docker Desktop
Скачайте и установите Docker Desktop для вашей ОС:
- **Windows**: https://desktop.docker.com/win/stable/Docker%20Desktop%20Installer.exe
- **Mac**: https://desktop.docker.com/mac/stable/Docker.dmg
- **Linux**: https://docs.docker.com/engine/install/

### 2️⃣ Запустите проект

Откройте терминал в папке проекта и выполните:

```bash
docker-compose up --build
```

**Или используйте Makefile:**

```bash
make up
```

### 3️⃣ Откройте в браузере

```
http://localhost:3000
```

### 4️⃣ Остановить проект

Нажмите `Ctrl+C` в терминале или выполните:

```bash
docker-compose down
```

**Или:**

```bash
make down
```
